## 🚀 Technologies
- NodeJS
- ReactJS
- React Hooks
- CSS

## ✂️ Clone Application
- ```git clone https://github.com/DaviQuental/guitarflix```
- ```cd guitarflix```

## ⚙️ Run Application with NPM
- ```npm install```
- ```npm start```

## ⚙️ Run Application with YARN
- ```yarn install```
- ```yarn start```
